Activity Log: (Aug.22.2016 -- Prior C videos posted)
 
 
Why C? Nearly every language has at its base a library written in C or C++.

C++ is based off C, though because of the changes found in C++, many things are no longer done the way they would've been done in C. It's kind of how FoxPro changed in moving from FoxPro for Windows to Visual FoxPro as C++ introduces the class to C, whereas C only has something called structures.

Still, the basic fundamentals are there. The same is true for C#, and many derivative languages (including Go and Java).

The following videos were made in 2013, and are for xbase developers wanting to learn C.
